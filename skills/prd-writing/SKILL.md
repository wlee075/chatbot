# PRD Writing System — Master Reference

This file is the single source of truth for system-wide constants, graph topology, PRDState fields, and the 13 PRD sections. All other skills in `skills/` cross-reference here rather than restating these facts.

## System Overview

A Streamlit + LangGraph chatbot that guides a PM through building a PRD section-by-section using a **reflection pattern**: ask 1 question → PM answers → draft section → reflect on 4 rubrics → PASS or loop.

**Stack:** Python 3.13, LangGraph `>=0.2.28`, Gemini Flash 2.5 (`gemini-2.5-flash`), Streamlit `>=1.32.0`, JSONL logging.

---

## Graph Topology

```
START
  │
load_context          ← emit welcome message; passthrough for context_doc
  │
generate_questions    ← Elicitor: asks 1 question; adds to chat_history
  │
await_answer          ← INTERRUPT: pauses graph; PM types in Streamlit chat input
  │
draft                 ← Drafter: synthesises all Q&A rounds into prose
  │
reflect               ← Reflector: scores 4 rubrics, emits VERDICT + TRIAGE
  │
  ├─ PASS / cap hit ──► advance_section ─┬─ more sections ──► generate_questions
  │                                      └─ all done      ──► finalize ──► END
  │
  └─ REWORK (iters remain) ──► generate_questions
```

**Key files:**

| Responsibility | File |
|---|---|
| Graph definition | `graph/builder.py` |
| All node functions | `graph/nodes.py` |
| Routing functions | `graph/routing.py` |
| State type | `graph/state.py` |
| All prompt templates | `prompts/templates.py` |
| Section definitions | `config/sections.py` |

## System-Wide Constants

Defined in `prompts/templates.py`. Import from there; never redefine.

| Constant | Value | Meaning |
|---|---|---|
| `PASS_SCORE_THRESHOLD` | `8.5` | Minimum OVERALL SCORE for `VERDICT: PASS` |
| `RECOVERY_MODE_SCORE_THRESHOLD` | `5.0` | OVERALL SCORE below this forces `TRIAGE: ENTER RECOVERY MODE` |
| `RECOVERY_MODE_RESOLVED_COMPONENT_THRESHOLD` | `0.5` | Resolved component % below this also triggers recovery |
| `DEFAULT_MAX_SECTION_ITERATIONS` | `5` | Max reflection loops per section before forced advance |
| `DEFAULT_MAX_CLARIFICATION_ATTEMPTS_PER_REQUIREMENT` | `3` | Max PM clarification attempts per gap |
| `DEFAULT_MAX_RECOVERY_MODE_CONSECUTIVE_ITERATIONS` | `2` | Consecutive recovery iterations before forced advance |

## PRDState Fields

Defined in `graph/state.py`.

**Session identity**

| Field | Type | Description |
|---|---|---|
| `thread_id` | `str` | Stable session identifier; scopes log files; set once per Streamlit session |
| `run_id` | `str` | UUID per `.invoke()` call; one per user answer roundtrip |

**Static configuration**

| Field | Type | Description |
|---|---|---|
| `context_doc` | `str` | Raw text of optional uploaded background document |
| `max_iterations` | `int` | Per-session override of `DEFAULT_MAX_SECTION_ITERATIONS` |

**Section navigation**

| Field | Type | Description |
|---|---|---|
| `section_index` | `int` | Index into `PRD_SECTIONS` (0–12) |
| `iteration` | `int` | Reflection loop count for the active section |

**Current section workflow**

| Field | Type | Description |
|---|---|---|
| `current_questions` | `str` | Last question generated by Elicitor |
| `section_qa_pairs` | `list` | `[{"questions": str, "answer": str}]` for all rounds this section |
| `current_draft` | `str` | Most recent draft from Drafter |
| `reflection` | `str` | Full reflection text from Reflector |
| `verdict` | `str` | `"PASS"` \| `"REWORK"` \| `""` |
| `requirement_gaps` | `str` | Extracted REQUIREMENT GAPS block from reflection |
| `triage_decision` | `str` | `"TRIAGE: ENTER RECOVERY MODE"` \| `"TRIAGE: NORMAL ITERATION"` |
| `recovery_mode_consecutive_count` | `int` | Consecutive ENTER RECOVERY MODE count; resets on PASS or NORMAL |
| `overall_score` | `float` | Parsed OVERALL SCORE; `-1.0` = parse failed |

**Accumulated output (reducers)**

| Field | Type | Reducer | Description |
|---|---|---|---|
| `prd_sections` | `dict` | `_merge_dicts` | `{section_id: draft_text}`; grows as sections pass |
| `chat_history` | `list` | `operator.add` | Append-only; each item has `role`, `type`, `content` |

**Final output**

| Field | Type | Description |
|---|---|---|
| `prd_markdown` | `str` | Compiled final PRD document |
| `is_complete` | `bool` | True when all 13 sections are done |

| 4 | `problem_statement` | Problem Statement | False |
| 5 | `goals` | Goals | False |
| 6 | `success_metrics` | Success Metrics | False |
| 7 | `non_goals` | Non-goals | False |
| 8 | `assumptions` | Assumptions & Validations | False |

| 10 | `proposed_solution` | Proposed Solution & Features | False |

| Field | Injected into |

## Section Reset Contract

`advance_section_node` resets all per-section state to initial values:

```python
"iteration": 0,
"verdict": "",
"reflection": "",
"current_draft": "",
"current_questions": "",
"section_qa_pairs": [],
"requirement_gaps": "",
"triage_decision": "",
"recovery_mode_consecutive_count": 0,
"overall_score": -1.0,
```

Fields that persist across sections: `thread_id`, `run_id`, `context_doc`, `max_iterations`, `section_index` (incremented), `prd_sections` (merged), `chat_history` (appended), `is_complete`.

---

## Human-in-the-Loop Interrupt

`await_answer_node` calls `interrupt(...)` from `langgraph.types`. The app resumes with:

```python
graph.invoke(Command(resume=user_input), config)
```

Each `.invoke()` generates a new `run_id`. `thread_id` is stable for the whole Streamlit session.

## Global Prompt Blocks

All defined in `prompts/templates.py`; injected by name into each agent prompt.

| Constant | Injected into | Purpose |
|---|---|---|
| `GLOBAL_RIGOR_BLOCK` | Elicitor, Drafter, Reflector | Enforce rigor; surface ambiguity explicitly |
| `DECISION_ENFORCEMENT_BLOCK` | Elicitor | All questions must be binary or option-choice |
| `ITERATION_DISCIPLINE_BLOCK` | Elicitor | Do not repeat resolved questions |
| `CONFIRMATION_RULE_BLOCK` | Clarification controller, Decision normalizer | What counts as a resolved PM response |
| `HUMAN_TRUST_BLOCK` | All agents | Anti-hallucination: "Not in knowledge bank" / "I do not know" |
| `SCORING_INTERPRETATION_BLOCK` | Reflector | Score band guide: 9–10 / 6–8.9 / <6 / <5 |
